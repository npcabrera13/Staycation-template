# 🏨 Admin Guide - Staycation Management System

## Overview
This guide explains how property owners/admins can manage their staycation business through the admin dashboard.

---

## 🔐 Accessing Admin Panel

### Login Flow
```
Landing Page → Admin Login (Footer) → Enter Password → Dashboard
```
- Scroll to the footer and find "Admin" link
- Enter your admin password
- Access the full management dashboard

---

## 📊 Admin Dashboard Sections

### 1. **Dashboard Overview**
- Total bookings count
- Revenue statistics
- Monthly/weekly trends
- Quick stats at a glance

### 2. **Bookings Calendar**
```
View Calendar → See Arrivals → Manage Bookings
```
- **All Upcoming Arrivals** - See who's arriving in the next 30 days
  - 🟢 Green = Arriving TODAY
  - 🟡 Yellow = Arriving Tomorrow
  - 🟠 Orange = This week
- **Grid View** - Monthly calendar with bookings
- **Timeline View** - Room-by-room occupancy

### 3. **Booking Management**
| Action | How |
|--------|-----|
| **Approve Booking** | Click booking → Confirm |
| **Cancel Booking** | Click booking → Cancel |
| **Edit Details** | Click booking → Modify info |
| **Delete** | Bulk select → Delete |

### 4. **Room Management**
```
Rooms Tab → Add/Edit/Delete Rooms
```
- **Add New Room**
  - Name, description, price
  - Capacity (max guests)
  - Amenities (WiFi, Pool, AC, etc.)
  - Gallery images (multiple URLs)
- **Edit Existing Room** - Update any details
- **Delete Room** - Remove from listings

### 5. **Site Settings**
```
Settings Tab → Customize Everything
```

| Setting | Purpose |
|---------|---------|
| **Brand Identity** | Site name and logo URL |
| **Theme Colors** | Primary and secondary colors |
| **Contact Info** | Phone, email, address |
| **Social Links** | Facebook, Instagram, TikTok |
| **Map Embed** | Google Maps embed code |
| **Payment Methods** | GCash and Bank Transfer QR codes |

### 6. **Visual Builder**
```
Visual Builder Button → Edit Live Website
```
- Click directly on text to edit
- Change hero images
- Modify colors in real-time
- Save & Publish changes

## 🔔 Managing Booking Statuses

### Workflow
```
Guest Books → Pending → Admin Reviews → Confirmed/Cancelled
```

1. **New bookings arrive as "Pending"**
2. **Review guest details and dates**
3. **Check for conflicts**
4. **Approve or decline**
5. **Contact guest if needed**

---

## 💰 Payment Setup

### GCash Setup
1. Go to Settings → Payment Methods
2. Enable GCash toggle
3. Enter account name and number
4. Upload QR code image URL

### Bank Transfer Setup
1. Enable Bank Transfer toggle
2. Enter bank name
3. Enter account name and number
4. Upload QR code image URL

### Messenger Link
- Add your Facebook Messenger link
- Guests click to confirm payment

## 📈 Best Practices

1. **Check Daily** - Review new bookings every morning
2. **Quick Response** - Approve/decline within 24 hours
3. **Update Calendar** - Keep availability accurate
4. **Quality Photos** - Use high-res room images
5. **Accurate Pricing** - Update prices for seasons
6. **Contact Info** - Keep phone/email current

## 🔒 Security Tips

1. **Strong Password** - Use a complex admin password
2. **Logout** - Always logout on shared devices
3. **Private Network** - Avoid public WiFi for admin tasks
4. **Regular Checks** - Monitor for unauthorized changes
