# 👤 User Guide - Staycation Booking System

## Overview
This guide explains how guests can discover, book, and manage their staycation reservations.

---

## 🏠 User Flow

### 1. **Discover the Property**
```
Landing Page → Browse Rooms → View Details
```
- Visit the website to see the hero section with property images
- Scroll down to view available rooms with photos, prices, and amenities
- Click on any room card to see detailed gallery and features

### 2. **Check Availability**
```
Select Room → Choose Dates → View Price
```
- Click "Book Now" or "View Details" on a room
- Select check-in and check-out dates
- See the calculated total price based on:
  - Number of nights
  - Room base price
  - Number of guests

### 3. **Make a Booking**
```
Fill Form → Submit → Receive Confirmation
```
- Enter your details:
  - Full name
  - Email address (important for lookup!)
  - Phone number
  - Number of guests
  - Special requests (optional)
- Review the booking summary
- Submit your reservation
- Receive confirmation with booking ID

### 4. **View Your Bookings**
```
My Bookings → Enter Email → View Status
```
- Click "My Bookings" in the navigation
- Enter the email used during booking
- View all your reservations with:
  - Booking status (Pending/Confirmed/Cancelled)
  - Check-in/out dates
  - Room details
  - Total price

| Feature | Description |
|---------|-------------|
| **Room Gallery** | Swipe through room photos |
| **Live Availability** | Real-time calendar showing booked dates |
| **Price Calculator** | Automatic total based on dates & guests |
| **Booking Lookup** | Find your booking using email |

| **Mobile Friendly** | Full functionality on any device |

---

## ⚠️ Important Notes

1. **Email is Key** - Use the same email to find your bookings later
2. **Pending Status** - Your booking needs admin approval before confirmation
3. **Payment** - Follow the payment instructions after booking (GCash/Bank Transfer)
4. **Cancellation** - Contact the property for cancellation requests

---

## 🔔 Booking Statuses Explained

| Status | Meaning |
|--------|---------|
| 🟡 **Pending** | Awaiting admin approval |
| 🟢 **Confirmed** | Booking approved - you're all set! |
| 🔴 **Cancelled** | Booking was cancelled |

---

## 💡 Tips for Users

1. **Book in Advance** - Especially for weekends and holidays
2. **Check Amenities** - Make sure the room has what you need
3. **Save Confirmation** - Screenshot or save your booking ID
4. **Contact Early** - Reach out with questions before booking
5. **Follow Up** - If pending for too long, contact the property
