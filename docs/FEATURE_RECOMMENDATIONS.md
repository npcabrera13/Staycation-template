# 🚀 Feature Recommendations

## High Priority (Recommended Next)

### 1. **Email Notifications**
- Send confirmation emails on booking
- Notify admin of new bookings
- Reminder emails before check-in
- **Why:** Improves communication and professionalism

### 2. **Payment Integration**
- Integrate PayMongo, GCash API, or Stripe
- Automatic payment status updates
- Receipt generation
- **Why:** Reduces manual payment verification

### 3. **Booking Calendar Sync**
- Export to Google Calendar
- iCal feed for availability
- Sync with Airbnb/Booking.com calendars
- **Why:** Prevents double-bookings

### 4. **Reviews & Ratings**
- Guest reviews after checkout
- Star rating system
- Admin moderation
- Display on room cards
- **Why:** Builds trust and credibility

---

## Medium Priority

### 5. **Seasonal Pricing**
- Weekend vs weekday rates
- Holiday surcharges
- Early bird discounts
- Long-stay discounts
- **Why:** Maximize revenue

### 6. **Promo Codes**
- Discount codes for marketing
- Track code usage
- Set expiry dates
- **Why:** Boost bookings during slow periods

### 7. **Multi-language Support**
- Filipino, English, Mandarin
- Auto-detect browser language
- **Why:** Reach international guests

### 8. **SMS Notifications**
- Booking confirmations via SMS
- Check-in reminders
- Payment reminders
- **Why:** Higher open rate than email

---
