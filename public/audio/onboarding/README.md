# Onboarding Voice Guide Premium Audio Files

This directory holds the high-fidelity human-like voice recordings (MP3s) used for the interactive **Admin Onboarding Guide** inside Paradise Resort.

## 🎵 How it Works

The onboarding guide checks this folder for specific filenames before falling back to the browser-native Text-to-Speech engine. By generating these files and placing them here, you get a gorgeous, 100% human studio experience for free.

## 🚀 Recommended Generation Settings
- **AI Voice Engine:** ElevenLabs (Adam, Rachel, or Antoni) or OpenAI TTS (Echo, Alloy).
- **Format:** MP3, mono, 44.1kHz.
- **Volume:** Uniform normalized gain.
